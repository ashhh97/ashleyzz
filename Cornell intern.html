Project 1: ScriptManager (On-going project)

ScriptManager is an opensource software written by JAVA for for analyzing sequencing data and generating figures. It contains 40 tools for genomics analysis. It includes graphical interface for non-programmer users and the command line interface. The product will lanuch in July. The feedback will be given by then.

My role

1.1_ScriptManager Icon Design（ongoing project）

The software now has 40 tools. But all the tools are described in the text. The lab assumed that we need icons to visualize the tools. I designed an icon for each of them.
