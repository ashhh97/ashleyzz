WETRY INTERN
05/05/2021-08/01/2021

‘WETRY’ is a fashion buying e-commerce platform that provides services including ‘try before you buy, fashion coordinator, and online shopping. 

MY ROLE

The incomplete return function makes the return process very inconvenient, which causes the customer's loss. I helped design a return and a post picture function. The return function is launched officially.

Feedback

Customers

The return process is terrible. It only has one return button. I cannot see the status of my order. I don't know where my money goes. I need to contact customers service to complete all the steps.

Aftersale Service

I Receive growing complaints about returns. The platform’s return system is imperfect, causing customer loss.

Company Finance

The imperfect return system leads to an unclear division of labor and chaotic management. I don't know when should I refund customers. 

Customer Service

I need to participate in every step of returning process, too much workload. 

User Journey Map

I create a user journey map to mark the steps that customers and the corresponding company department need to go through and find their paint point. After generating this diagram. I have a basic understanding the return process workflow I need to map out.
